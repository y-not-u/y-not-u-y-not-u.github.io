Lissh基于Alfred3的快捷ssh连接

2018年4月9日

灵感来自于Shuttle，定义快捷命令，提供轻量级的命令执行。
主要是使用Alfred的workflow来管理和连接一些少量主机。

Github

https://github.com/VoganWong/Lissh

Features

  • 支持添加 sshadd {host} {cmd}
  • 支持删除 sshdel {host}
  • 支持列表选择/筛选
  • 支持文件编辑 sshopen
